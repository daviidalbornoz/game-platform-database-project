# Game Platform Database Project

This project demonstrates the design and implementation of a relational database using SQL Server.

The project was completed as part of a Database Design and Administration course and includes the full lifecycle of database development, from conceptual modeling to query execution.

---

## Project Components

### 1. Entity Relationship Diagram (ERD)
Design of the database structure showing entities and relationships.

📁 Folder: `ERD`

---

### 2. Relational Schema
Conversion of the ERD into relational tables.

📁 Folder: `schema`

---

### 3. Physical Database Design
Definition of table attributes, data types, and constraints.

📁 Folder: `physical_design`

---

### 4. Table Creation
Implementation of the database tables in SQL Server Management Studio.

📁 Folder: `table_creation`

---

### 5. Data Population
Insertion of records into the database tables.

📁 Folder: `data_population`

---

### 6. SQL Objects
Creation of advanced SQL components including:

- Views
- Stored Procedures
- User-defined Functions

📁 Folder: `sql_objects`

---

### 7. Query Execution
Execution of SQL queries to analyze and retrieve information from the database.

📁 Folder: `queries_proof`

---
